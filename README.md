# 🏦 ATM Transaction Analysis System

A Python-based ATM Banking Simulation System that allows users to register, login, perform transactions, and analyze financial data using Pandas.

---

## 🚀 Features

- 🔐 Secure User Registration & Login (4-digit PIN authentication)
- 💰 Deposit & Withdrawal functionality
- 📜 Transaction History tracking
- 📊 Mini Statement (Last 5 transactions)
- 📈 Monthly Transaction Analysis using Pandas
- 🕒 Timestamp-based transaction recording

---

## 🛠 Technologies Used

- Python
- Pandas
- Datetime
- Getpass
- Git & GitHub

---

## 📊 Data Analysis Capabilities

The system performs:

- ✔ Total Deposited Amount Calculation
- ✔ Total Withdrawn Amount Calculation
- ✔ Monthly Transaction Summary using GroupBy
- ✔ DataFrame-based transaction analysis
